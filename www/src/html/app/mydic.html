MyDic

OS：Android
Price：free
Latest Version：1.0.1

About this application

Mydic creates your own dictionary.
And you can play with the dictionary you create.

How to use this application

1: You register a dictionary in the application.
2: Add a word to the dictionary.
3: If there are some words, you can play.
